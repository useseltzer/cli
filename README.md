# CLI

[![NPM version][npm-image]][npm-url]
[![Downloads][downloads-image]][npm-url]

## Installation

```bash
pnpm i -g @seltzer/cli
```

```bash
npm i -g @seltzer/cli
```

## Getting Started

```bash
seltzer start
```
or
```bash
seltzer start <database_url>
```
